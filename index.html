Everything You Need to Know

Get answers to common questions about Chat Navigator

General

It adds a side-panel to ChatGPT so you can see an outline of every message, jump instantly, search/filter, bookmark entire chats or single messages, auto-extract code blocks, and reuse a tagged prompt library.

Chat Navigator works inside the current conversation: it shows a live outline, precise filters (You/AI/★/Code), dual-level bookmarks, and one-click code collection. It's purpose-built for long, messy threads where scrolling fails.

It's lightweight. On huge threads, the outline and search index may use some memory while the tab is open. You can disable features like auto-code extraction if you need to minimize overhead.

Technical

Works on Chromium-based browsers that support Chrome extensions (Chrome, Brave, Edge, Arc). Firefox isn't supported yet.

Yes, it runs on chat.openai.com in most org setups. If your company enforces strict content-security settings, you may need to allow the extension—ask your admin.

Common fixes:

  • Refresh the ChatGPT tab after installing/updating the extension.
  • Ensure you're on chat.openai.com and signed in.
  • Pin the extension and click the Chat Navigator button to activate.
  • Temporarily disable conflicting UI extensions and reload.
